# === jargon ===
1. GenAI : an AI system that can generate content, like text, video...
2. NLP (natural language processing) : a field used to generate language, some concrete model like rnn, lstm, llm...
3. LLM (large language model) : a method(model) in the NLP, usually based on the transformer, like GPT...
4. tranformer : an architecture proposed in 2017 

# === ML vs DL ===
- ml needs to choose the characters manually, and DL not.
